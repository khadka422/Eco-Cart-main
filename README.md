# Eco Cart

Eco Cart is an eco-friendly e-commerce platform built to promote sustainable shopping. The project includes a modern frontend, a backend server, MongoDB database integration, and an AI chatbot powered by Ollama.

The goal of Eco Cart is to provide users with a smooth online shopping experience while encouraging environmentally friendly product choices.

---

# Features

- 🌱 Eco-friendly shopping platform
- 🛒 Add to cart functionality
- 🔐 User authentication system
- 📦 Product management
- 🤖 AI chatbot integration using Ollama
- 📱 Responsive design for all devices
- ⚡ Fast and scalable backend

# Step-by-Step Setup Guide

## Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/eco-cart.git
```

---

## Step 2: Move into the Project Folder

```bash
cd eco-cart
```

---

## Step 3: Install Project Dependencies

Before running the project, install all required packages:

```bash
npm install
```

---

## Step 4: Start the Frontend

Run the frontend development server:

```bash
npm run dev
```

After running the command, the frontend will start on:

```bash
http://localhost:3000
```

---

## Step 5: Start the Backend Server

Open a new terminal and run:

```bash
nodemon start
```

This will start the backend server.

---

## Step 6: Start MongoDB

Make sure MongoDB is running before using the application:

```bash
mongod
```

---

## Step 7: Run the AI Chatbot

To use the AI chatbot feature, start Ollama using:

```bash
ollama run llama3
```

Make sure Ollama is installed properly on your system.
